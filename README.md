# Xhc Manuscript

This repository contains additional codes and scripts associated with the following manuscript:

Rahman A, Wagner N, Scott JC, Dung JKS, Schiffer AM, Pupko T, White RC, Savory EA, du Toit LJ, Grünwald NJ, Weisberg AJ, Chang JH.  **Evolution and global spread of Xanthomonas hortorum pv. carotae in the context of agricultural modernization**

- `src` Contains R scripts for calculating pangenome statistics and phlowave analysis.
- `data` Contains associated metadata, tree files, and R functions for those scripts.
- `additional_results` Contains individual timetree figures for Xhc clades.
